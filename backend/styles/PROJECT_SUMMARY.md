## 🆘 Getting Help

### If Components Don't Show:
1. Ensure `npm run build` was run
2. Check `styles/design-system.css` is imported
3. Clear browser cache

### If Responsive Doesn't Work:
1. Add `<meta name="viewport">` to HTML head
2. Use correct breakpoint prefixes (md:, lg:, xl:)
3. Design mobile-first

### If Dark Mode Doesn't Work:
1. Add `class="dark"` to `<html>` tag
2. All components have `dark:` prefixed styles
3. Test in browser DevTools
