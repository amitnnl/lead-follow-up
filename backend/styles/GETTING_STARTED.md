## 🆘 QUICK TROUBLESHOOTING

### Components not styled?
```
1. npm run build (rebuild)
2. Check design-system.css is imported
3. Clear browser cache (Ctrl+Shift+Delete)
```

### Responsive not working?
```
1. Add <meta name="viewport"> to <head>
2. Use md:, lg:, xl: prefixes
3. Design mobile-first (no prefix = mobile)
```

### Dark mode not working?
```
1. Add class="dark" to <html> tag
2. All components have dark: prefixes
3. Test in DevTools
```
